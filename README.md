# Yearbook 연감
YEARBOOK 연감 is a digital yearbook celebrating our class memories, friendships, and milestones. Explore photos, personal stories, events, and messages for teachers and classmates, all in one interactive and beautifully designed online space.

---

## Overview 개요

The Yearbook web project is a personal, front-end focused project that captures the spirit of a school year in a clean, interactive, and user-friendly website.  

Objectives 목표:   
- Preserve school memories digitally in a structured format  
- Provide a polished and interactive browsing experience  
- Develop and demonstrate web design and front-end development skills  

---

## Features 기능

- Responsive layout compatible with desktop, tablet, and mobile devices  
- Interactive navigation through different sections of the yearbook  
- Organized photo galleries for school events, trips, and activities  
- Event highlights with text summaries and photos  
- Clean and modern design with clear typography and layout  
- Easy maintenance and content updates
